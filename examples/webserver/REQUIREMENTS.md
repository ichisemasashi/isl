# webserver 要件定義 (v1.0)

この文書は `examples/webserver` の固定要件です。  
実装・テスト・運用は本要件に従います。

## 1. 固定要件

| 項目 | 固定値 |
| --- | --- |
| 対応プロトコル | HTTP/1.1 のみ (TCP) |
| 対象OS | Linux (x86_64) のみ |
| リッスンポート | `8080` 固定 |
| 同時接続数 | 最大 `100` 接続 |
| 静的配信の範囲 | ドキュメントルート配下の `/public/*` のみ |
| CGI | 非対応 (`CGI` / `FastCGI` ともに未サポート) |

## 2. 補足

- HTTPS 終端は本 webserver の責務外とし、必要時はリバースプロキシで実施する。
- `8080` 以外のポート対応は v1.0 のスコープ外。
- 静的配信は通常ファイルのみを対象とし、`/public` 外へのパス遡り (`..`) は禁止する。
- 本要件の変更は、本ファイル更新を必須とする。
