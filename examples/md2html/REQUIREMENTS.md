# md2html 要件定義 (v0.1)

最終更新: 2026-02-16

## 1. 目的

`md2html` は Markdown テキストを HTML へ変換するコマンドラインツールとする。

## 2. 入力要件

- 入力は次のいずれか:
  - `*.md` ファイルを 1 つ指定
  - 標準入力 (`stdin`) から Markdown テキストを受け取る
- ファイル入力と標準入力の同時指定は不可。
- 文字コードは UTF-8 を前提とする。

## 3. 出力要件

- 変換結果の HTML は標準出力 (`stdout`) へ出力する。
- 変換エラーは標準エラー出力 (`stderr`) に出力する。
- 出力ファイル作成オプション (`-o` など) は v0.1 では提供しない。

## 4. 変換方式要件

- 変換エンジンは **内部実装** とする。
- `pandoc` を含む外部 Markdown 変換コマンドには依存しない。
- 必要最小限の Markdown 構文を段階的に実装する。

## 5. CLI 仕様 (v0.1)

- 想定実行形式:
  - `md2html path/to/input.md`
  - `cat input.md | md2html`
- 引数なしで標準入力が空の場合はエラー終了する。

## 6. 終了コード

- `0`: 正常終了
- `1`: 使い方エラー（引数不正など）
- `2`: 入力エラー（ファイル不存在/読込失敗など）
- `3`: 変換エラー（内部変換失敗）

## 7. スコープ外 (v0.1)

- ディレクトリ一括変換
- CSS テンプレート適用
- 目次自動生成
- 外部リンク検証
