# Top Page Settings

## 概要

`/wiki` の表示先は、次の優先順位で決まります。

1. ユーザー個別トップページ
2. システム・デフォルトのトップページ
3. 従来のトップページ（ページ一覧）

個別トップページが未指定ならシステム・デフォルトを使います。  
システム・デフォルトも未指定なら、従来どおり `/wiki` のページ一覧を表示します。

指定先のページが削除済み、または現在のユーザー権限では表示できない場合も、次の候補へフォールバックします。

## 管理画面

- システム・デフォルト: `/wiki/admin/settings`
- ユーザー個別: `/wiki/admin/users`

どちらも slug を空欄にすると設定解除です。

## 従来トップページの機能を出す方法

システム・デフォルトのトップページに、従来 `/wiki` が表示していた機能を埋め込みたい場合は、対象ページの Markdown 本文に次のトークンを書きます。

```md
{{wiki-home-features}}
```

このトークンは、表示時に次のブロックへ展開されます。

- 既存のページ一覧
- editor / admin 向けの `Create New Page`, `Add Media`, `Article Management` リンク

## 使い方の例

```md
# Portal

ようこそ。

{{wiki-home-features}}
```

このページを `/wiki/admin/settings` でシステム・デフォルトのトップページに設定すると、`/wiki` でポータル本文と従来トップページの機能をあわせて表示できます。
