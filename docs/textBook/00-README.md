# ISLISP プログラミング教科書シリーズ

本シリーズは ISLisp 処理系 **isl** を題材に、ISLISP プログラミングを習熟度順に学ぶための教科書群です。

## 構成方針

二層 × スパイラルの構成を採用しています。

- **第1層（共通幹 / `part1-core/`）**: 習熟度順の一本道。全員必修。
- **第2層（専門枝 / `part2-tracks/`）**: 処理系特性ごとの専門編。番号順が推奨読破順だが、関心ある編へ飛んでもよい。
- **スパイラル**: 1 つの題材 **数式インタプリタ "isl-calc"** が、各段階で技術を変えながら作り直され成長していく。同じものを繰り返し作ることで定着を図る（内容の重複は意図的）。

## 貫く題材: 数式インタプリタ "isl-calc"

`(+ 1 (* 2 3))` のような式を評価する小さなインタプリタを、REPL での一行評価から始め、オブジェクトシステムによる AST 設計、規格準拠版、Web サービス化、ネイティブコンパイル、そして処理系内部の観察まで、7 周にわたって育てていきます。「プログラム＝データ」「eval」という Lisp の核心を一本の物語で体感するのが狙いです。

## スパイラル周回マップ

| 周 | 章 | isl-calc の姿 | 解放される処理系機能 |
|----|----|--------------|--------------------|
| 0 | `part1-core/01-introduction` | REPL で `(+ 1 2)` を評価する | `./bin/isl`・REPL・S式 |
| 1 | `part1-core/02-basic` | リスト表現＋再帰で四則電卓 | リスト操作・再帰・条件分岐 |
| 2 | `part1-core/03-intermediate` | オブジェクトシステムで AST をモデル化、変数・関数定義 | オブジェクトシステム・高階関数・エラー基礎 |
| 3 | `part2-tracks/01-standard` | strict プロファイルで規格準拠版に | strict・`with-handler`・パッケージ |
| 4 | `part2-tracks/02-extended` | 式評価 Web サービス＋履歴永続化 | 拡張プリミティブ・I/O・ネットワーク |
| 5 | `part2-tracks/03-compiler` | 式をネイティブ計算（AOT/JIT） | `islc`・emit-llvm・IR・JIT |
| 6 | `part2-tracks/04-internals` | core.scm の評価器と自作評価器を比較 | `core.scm`・評価ループ・ABI |

## 読み方

1. まず `part1-core/` を 01 → 02 → 03 の順に通読する（全員必修）。
2. その後 `part2-tracks/` を 01 → 02 → 03 → 04 の順に進む。目的が明確なら関心ある編へ飛んでよい。
3. 各章末尾の「★ isl-calc を作り直す」節がスパイラルの作り直し回。前の周の自分のコードを思い出しながら新しい技術へ置き換える。

## 動作環境

- 実装言語: Gauche 0.9.15（Scheme 処理系）
- REPL 起動: `./bin/isl`
- ファイル実行: `./bin/isl examples/hello.lsp`
- strict プロファイル: `./bin/isl --profile strict <file>`

詳細はリポジトリ直下の `CLAUDE.md` および `docs/` 配下の各仕様書を参照。
