# 3.1 高階関数

> スパイラル 2 周目 / 中級編 第 1 章

## 学習目標
- 関数を引数に取る／返す関数を書ける。
- `mapcar` などで反復をデータ駆動に書ける。

## 前提知識
- 基本編（関数定義・リスト・再帰）を読了。

## 内容（アウトライン）
- 関数を値として渡す（`#'` / `funcall` / `apply`）。
- `mapcar` による一括変換。
- オペランド列の一括評価（電卓の多引数化への布石）。

## サンプル
```lisp
(mapcar (lambda (x) (* x x)) '(1 2 3))   ; => (1 4 9)
```

## 演習
- 引数リスト全体を `calc-eval` に通すヘルパを書く。
