# 3.2 クロージャ

> スパイラル 2 周目 / 中級編 第 2 章

## 学習目標
- レキシカルスコープと自由変数の捕捉を理解する。
- クロージャで状態（環境）を持ち回せる。

## 前提知識
- 高階関数（[3.1](01-higher-order.md)）。

## 内容（アウトライン）
- `lambda` が定義時の環境を捕捉すること。
- カウンタなど状態を持つクロージャ。
- 変数環境（名前→値の対応）をクロージャ／連想リストで表す ＝ 次章以降の「変数を持つ電卓」への布石。

## サンプル
```lisp
(defun make-adder (n)
  (lambda (x) (+ x n)))
(funcall (make-adder 10) 5)   ; => 15
```

## 演習
- 変数環境を連想リストで表し、`(lookup name env)` を書く。
