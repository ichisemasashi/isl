# 1.1 ISLISP とは

> スパイラル 0 周目 / 入門編 第 1 章

## 学習目標
- ISLISP がどんな言語かを一言で説明できる。
- 本書で使う処理系 isl の位置づけを理解する。

## 前提知識
- なし（プログラミング未経験でも読める）。

## 内容（アウトライン）
- ISLISP は ISO/IEC 13816 で標準化された Lisp 系言語であること。
- 「プログラム＝データ（S 式）」という Lisp の基本思想。
- 本書の処理系 isl は Gauche 上に実装されたインタプリタ＋コンパイラであること。
- strict（標準準拠）と extended（拡張）の 2 プロファイルがあること（詳細は後の編）。

## 本書の題材「isl-calc」
本書は 1 つの数式インタプリタ `isl-calc` を 7 周にわたり育てる。ここではゴール像だけ示す。
