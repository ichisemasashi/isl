# 2.3 再帰

> スパイラル 1 周目 / 基本編 第 3 章

## 学習目標
- 基底ケースと再帰ケースに分けて関数を書ける。
- リストや木構造を再帰でたどれる。

## 前提知識
- リスト操作（[2.2](02-list-as-expression.md)）。

## 内容（アウトライン）
- 再帰の基本形（基底ケース + 自己呼び出し）。
- リストの長さ・合計を再帰で求める。
- 入れ子の式（木）を再帰でたどる発想 ＝ 次章の評価器の核心。
- 末尾再帰への言及（深入りは中級編）。

## サンプル
```lisp
(defun sum-list (xs)
  (if (null xs)
      0
      (+ (car xs) (sum-list (cdr xs)))))
```

## 演習
- ネストした数値リストの要素数を数える関数を書く。
