# 2.2 リストで式を表す

> スパイラル 1 周目 / 基本編 第 2 章

## 学習目標
- `cons` / `car` / `cdr` / `list` でリストを構築・分解できる。
- 数式 `(+ 1 2)` を「演算子＋オペランドのリスト」として捉えられる。

## 前提知識
- 変数束縛と関数定義（[2.1](01-binding-and-functions.md)）。

## 内容（アウトライン）
- リストの構築 `(list '+ 1 2)` と分解 `car` / `cdr`。
- 式の先頭＝演算子、残り＝オペランド、という見方。
- クオートされた式 `'(+ 1 (* 2 3))` を data として走査する。
- `null` 判定とリストの末尾。

## サンプル
```lisp
(defglobal expr '(+ 1 2))
(car expr)        ; => +
(cdr expr)        ; => (1 2)
```

## 演習
- 式 `(* 3 4)` から演算子と 2 つのオペランドを取り出して表示する。
