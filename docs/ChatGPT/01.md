## **第1部: ISLISP入門**  
### **1. ISLISP入門** 

#### **ISLISPとは？**
ISLISPは、プログラミングのためのシンプルかつ強力な環境を提供するために設計された、Lispプログラミング言語の最小限の機能を持つ方言です。 実装を軽量かつ使いやすく保ちつつ、既存の多くのCommon Lisp機能との互換性を維持することを目的としています。 

ISLISPの主な特徴は以下の通りです。

- **小さなフットプリント：** コンパクトで高速に動作するように設計されており、組み込みシステムやリソースに制約のある環境に最適です。
- **インタプリタ言語：** ISLISPは通常、インタプリタ言語として使用され、コードの開発やテストを対話的に行うことが容易になります。
- **互換性：** Common Lispと多くの機能が共通していますが、その複雑なシステムの一部を簡素化し、新規ユーザーにとってより利用しやすくなっています。

#### **ISLISP環境の設定**
ISLISPでのプログラミングを始めるには、ISLISPインタープリターをインストールする必要があります。一般的に使用されている実装の1つに、オープンソースのISLISP実装である**EISL**があります。

**ISLISP（EISL経由）の設定手順**

1. **ISLISP（EISL）をダウンロードしてインストールする**  
   [EISL GitHub リポジトリ](https://github.com/sasagawa888/eisl) にアクセスします。次のコマンドを使用してクローンを作成できます。

   ```bash
   git clone https://github.com/sasagawa888/eisl.git
   ```

2. **EISL のビルド**  
   リポジトリをクローンした後、プロジェクトディレクトリに移動し、インストール手順に従います（手順はご利用のオペレーティングシステムによって異なります）。
   - Unix系システムの場合は、通常、以下のようにしてビルドできます。
     ```bash
     make
     ```
   - その他のシステムの場合は、プラットフォーム固有の手順についてはリポジトリのREADMEファイルを参照してください。

3. **ISLISPの実行**  
   インストールが完了したら、ターミナルで次のコマンドを実行して ISLISP REPL (Read-Eval-Print Loop) を起動します。

   ```bash
   ./eisl
   ```

   これにより ISLISP インタプリタが起動し、Lisp 式の入力と評価を開始できます。



#### **初めての ISLISP プログラムの実行**

すべてが正しく設定されていることを確認するために、コンソールに「Hello, World!」と出力する簡単な ISLISP プログラムを実行してみましょう。

1. ISLISP REPL を開きます。
2. 次のコードを入力します。

   ```lisp
   (format t "Hello, World!")
   ```

3. **Enter**キーを押すと、次のような出力が表示されます。

   ```
   Hello, World!
   ```

この基本的なプログラムは、ISLISPで画面に表示するために使用される**`format`**関数の使用方法を示しています。 `format`の`t`引数は、出力を標準出力（すなわち、ターミナル）に送信すべきであることを示しています。
