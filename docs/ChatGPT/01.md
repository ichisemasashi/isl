# 1. ISLISP入門（標準仕様ベース）

## ISLISPとは
ISLISPは ISO/IEC 13816 で標準化された Lisp 方言です。設計目標は以下です。

- 既存Lispとの実用的互換性
- 必要十分な中核機能
- オブジェクト指向機能の統合
- 拡張可能性
- 実装効率

## 真偽値の基本
- `t` は真
- `nil` は偽かつ空リスト
- 条件式では `nil` だけが偽（それ以外は真）

## この文書での方針
- 標準仕様で規定される機能を優先
- 実装依存機能（FFI、DB、ネットワーク、パッケージ拡張など）は「拡張」と明記

## 最初の例
```lisp
(defun fact (n)
  (if (= n 0)
      1
      (* n (fact (- n 1)))))

(fact 5) ; => 120
```
