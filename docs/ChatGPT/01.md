## 1.1 ISLISPの位置づけ
ISLISPは ISO/IEC 13816 で標準化された Lisp 言語である。標準は「適合処理系」と「適合プログラム（text）」を定義し、実装依存拡張を許容する。

## 1.2 適合性（compliance）
標準の観点で重要なのは次の2点。
- 処理系が必須仕様を満たすこと
- プログラムが規格で未定義/実装依存領域に過剰依存しないこと

## 1.3 記法上の約束（本書）
- 評価結果は `; => ...` で示す
- 標準外機能は「拡張」と明示する

## 1.4 最小例
```lisp
(defun fact (n)
  (if (= n 0)
      1
      (* n (fact (- n 1)))))

(fact 5) ; => 120
```

## 1.5 真偽値
```lisp
(if nil 'a 'b) ; => b
(if t   'a 'b) ; => a
```
`nil` のみが偽で、それ以外は真として扱う。
