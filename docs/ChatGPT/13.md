# 13. 小規模アプリケーション設計

## 方針
- 入力/処理/出力を分離
- 失敗時の経路を先に設計
- 標準機能で書ける部分を増やす

## 最小例
```lisp
(defun process-line (s)
  (format nil "INPUT=~A" s))

(defun main ()
  (with-open-file (in "input.txt" :direction :input)
    (with-open-file (out "output.txt" :direction :output :if-exists :supersede)
      (let ((line (read-line in nil)))
        (while line
          (write-line (process-line line) out)
          (setq line (read-line in nil)))))))
```

## 注意
コマンドライン引数取得APIは標準で固定されないため、処理系仕様に合わせること。
