# 3. 制御構造と関数

## 条件分岐
### if
```lisp
(if (> 5 3) 'yes 'no)
```

### cond
```lisp
(cond
  ((< x 0) 'neg)
  ((= x 0) 'zero)
  (t 'pos))
```

### case
```lisp
(case x
  ((1) 'one)
  ((2 3) 'small)
  (otherwise 'other))
```

## 反復
ISLISPでは `while`, `do`, `dolist`, `dotimes` などが利用される（処理系差あり）。

## 関数定義
```lisp
(defun square (n) (* n n))
```
