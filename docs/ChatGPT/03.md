# 3. 制御構造と関数

## 対応節
- §12 Forms and evaluation
- §14 Control structure

## 3.1 条件分岐
```lisp
(if (> 3 2) 'yes 'no) ; => yes

(cond
  ((< x 0) 'neg)
  ((= x 0) 'zero)
  (t 'pos))

(case x
  ((1) 'one)
  ((2 3) 'small)
  (otherwise 'other))
```

## 3.2 反復
```lisp
(while (< i 10)
  (setq i (+ i 1)))

(do ((i 0 (+ i 1)))
    ((= i 10) i)
  (print i))
```

## 3.3 非局所制御
```lisp
(block exit
  (return-from exit 42)
  0) ; => 42

(catch 'tag
  (throw 'tag 'ok)) ; => ok
```

## 3.4 ジャンプ制御
`tagbody` と `go` は構造化されない制御を提供する。可読性のため最小限に使う。
