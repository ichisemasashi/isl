# 5. リストとシーケンス

## 対応節
- §21 List class
- §25 Sequence functions

## 5.1 リスト基本
```lisp
(car '(1 2 3)) ; => 1
(cdr '(1 2 3)) ; => (2 3)
(cons 0 '(1 2 3)) ; => (0 1 2 3)
```

## 5.2 走査・変換
```lisp
(mapcar (lambda (x) (* x 2)) '(1 2 3)) ; => (2 4 6)
(remove-if oddp '(1 2 3 4 5)) ; => (2 4)
(remove-if-not oddp '(1 2 3 4 5)) ; => (1 3 5)
```

## 5.3 探索・集約
```lisp
(find 3 '(1 2 3 4)) ; => 3
(reduce + '(1 2 3 4)) ; => 10
```

## 5.4 実装指針
- 前から順に処理: リストが自然
- インデックス中心: ベクタ/配列を検討
