# 2. 基本構文とデータ型

## 基本評価規則
- アトムはその値として評価される（シンボルは束縛を参照）
- リストは通常「関数呼び出し」
- `'x` は `(quote x)` の糖衣構文

## 主要データ
- シンボル
- 数値
- 文字、文字列
- consセル/リスト
- ベクタ/配列

## リストと cons
```lisp
(cons 1 (cons 2 (cons 3 nil))) ; => (1 2 3)
(car '(1 2 3)) ; => 1
(cdr '(1 2 3)) ; => (2 3)
```

## 文字列と文字
```lisp
"Hello"
#\A
```

## 変数束縛
- ローカル: `let`, `let*`
- 大域: 処理系提供の定義子（例: `defglobal`, `defvar` など）は実装差がある

```lisp
(let ((x 10) (y 20))
  (+ x y)) ; => 30
```
