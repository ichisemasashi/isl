## 2.1 字句と表現
- シンボル、数値、文字列、文字、括弧、引用などの字句規則は §6-7 に定義される。
- `'x` は `(quote x)`。

## 2.2 評価モデル（概要）
- シンボルは束縛を参照
- リストは通常「関数呼び出しフォーム」
- 特殊形式は個別規則で評価

## 2.3 シンボルと数値
```lisp
'foo ; => foo
(+ 1 2 3) ; => 6
```

## 2.4 文字と文字列
```lisp
#\A
"Hello"
```

## 2.5 consとリスト
```lisp
(cons 1 (cons 2 nil)) ; => (1 2)
(car '(a b c)) ; => a
(cdr '(a b c)) ; => (b c)
```

## 2.6 配列・ベクタ・文字列
```lisp
(make-array 3)
(vector 1 2 3)
(length "abc") ; => 3
```

## 2.7 用語統一
- 「リスト」: consセル連鎖で末尾が `nil`
- 「ベクタ」: インデックスアクセスを持つ線形列
- 「配列」: 規格上の配列クラス（実装は1次元中心のこともある）
