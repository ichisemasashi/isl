# 9. 条件処理とデバッグ

## 対応節
- §9 Errors
- §14 Control structure
- §29 Condition system

## 9.1 非局所脱出
```lisp
(catch 'exit
  (throw 'exit 99)) ; => 99
```

## 9.2 条件分岐
```lisp
(handler-case
  (/ 1 0)
  (error (e) 'failed))
```

## 9.3 エラー分類
規格は、エラー仕様のクラスと、処理系依存となる領域を区別している。移植コードでは「未定義動作に依存しない」ことを優先する。

## 9.4 デバッグ
`trace`, `break`, `debug` 等は便利だが、標準での必須度は低く実装差がある。
