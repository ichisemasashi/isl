# 8. ストリームと入出力

## 対応節
- §26 Stream class
- §27 Input and output
- §28 Files

## 8.1 読み取り
```lisp
(with-open-file (s "input.txt" :direction :input)
  (read-line s nil))
```

## 8.2 書き込み
```lisp
(with-open-file (s "output.txt" :direction :output :if-exists :supersede)
  (format s "~A~%" "hello"))
```

## 8.3 代表関数
- `read`, `read-line`
- `format`, `write-line`
- `with-open-file`

## 8.4 注意
ファイルオープンキーワード（`:if-does-not-exist` 等）の受理範囲は実装差がある。処理系マニュアルを併読すること。
