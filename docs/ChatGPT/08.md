# 8. ストリームと入出力

## ファイルI/O
```lisp
(with-open-file (s "data.txt" :direction :output :if-exists :supersede)
  (format s "~A~%" "hello"))

(with-open-file (s "data.txt" :direction :input)
  (read-line s nil))
```

## 代表的操作
- `read` / `read-line`
- `format`
- `write-line`
- `probe-file` / `delete-file`（処理系拡張の場合あり）

## 注意
`with-open-file` のキーワード群（`:if-does-not-exist` など）は実装差があるため、処理系仕様を確認すること。
