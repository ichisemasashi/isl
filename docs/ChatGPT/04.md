# 4. 関数と高階プログラミング

## 対応節
- §12 Forms and evaluation
- §25 Sequence functions

## 4.1 関数定義
```lisp
(defun add (x y) (+ x y))
(add 3 4) ; => 7
```

## 4.2 ラムダ式
```lisp
((lambda (x) (* x x)) 5) ; => 25
```

## 4.3 関数を値として扱う
```lisp
(mapcar (lambda (x) (+ x 1)) '(1 2 3)) ; => (2 3 4)
(reduce + '(1 2 3 4)) ; => 10
```

## 4.4 可変引数・適用
```lisp
(apply + '(1 2 3 4)) ; => 10
```

## 4.5 標準準拠上の注意
Common Lisp 固有記法（例: `#'`）は ISLISP 標準の中心機能ではない。移植性を重視するなら、処理系で仕様確認する。
