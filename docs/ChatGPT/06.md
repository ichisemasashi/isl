# 6. シンボル、束縛、スコープ

## スコープ
ISLISPは字句スコープを基本とします。`let`/`let*` で局所束縛を作ります。

```lisp
(let ((x 10))
  (let ((f (lambda (y) (+ x y))))
    (funcall f 5))) ; => 15
```

## 大域束縛
大域変数や定数の導入形式は処理系差があります。

## 重要な注意（標準と拡張）
- `defpackage`/`in-package` などのパッケージ機能は、ISLISP標準外として扱う実装があります。
- 名前空間管理機構は処理系拡張の仕様を必ず確認してください。
