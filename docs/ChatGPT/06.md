# 6. シンボル、束縛、スコープ

## 対応節
- §11 Scope and extent
- §12 Forms and evaluation
- §18 Symbol class

## 6.1 束縛と有効範囲
```lisp
(let ((x 10))
  (let ((f (lambda (y) (+ x y))))
    (funcall f 5))) ; => 15
```

## 6.2 代表的束縛形式
- `let`: 並列束縛
- `let*`: 逐次束縛
- `lambda`: 関数束縛

## 6.3 extent（存続期間）
規格は、束縛のスコープ（可視範囲）と extent（存続）を区別して扱う。

## 6.4 標準外機能の扱い
`defpackage`/`in-package` などの名前空間拡張は、標準仕様の核ではない。実装依存として扱う。
