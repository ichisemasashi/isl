# 12. 外部連携（標準外拡張）

## 対応節
- 標準本体では直接規定されない（参考: §30 Miscellaneous）

## 12.1 FFI
処理系提供APIでC関数を呼ぶ。型表現・呼出規約・エラー処理は実装依存。

## 12.2 OS連携
`system`、環境変数、プロセス制御は拡張であることが多い。

## 12.3 DB/ネットワーク
標準仕様では API を規定しない。外部ライブラリ/実装機能で提供される。

## 12.4 移植指針
- 外部連携コードを隔離
- フォールバック経路を用意
- 拡張依存表をREADMEに明記

## 12.5 注意（誤解防止）
「ISLISP標準が Common Lisp ライブラリ直接利用を保証する」わけではない。相互運用可否は処理系ごとに異なる。
