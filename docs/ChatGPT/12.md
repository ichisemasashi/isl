# 12. 外部システム連携（拡張）

この章の内容は原則として **ISLISP標準外** です。

## 1. FFI（拡張）
処理系が提供する FFI を使って C 関数を呼び出せます。APIは処理系依存です。

## 2. Common Lisp との相互運用
「ISLISP標準として直接相互運用を規定」はされません。可能かどうかは実装依存です。

## 3. OS連携
`system`, 環境変数取得、プロセス制御などは多くが拡張機能です。

## 4. DB/ネットワーク
標準仕様では直接規定されません。ライブラリまたは処理系拡張で提供されます。

## 実務上の指針
- 移植性重視: 標準コア + 抽象化レイヤ
- 実装活用: 拡張APIを明示的に隔離
