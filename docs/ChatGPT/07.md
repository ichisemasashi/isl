# 7. オブジェクトシステム

## 基本
ISLISPはオブジェクトシステムを持ち、`defclass`/`make-instance`/`defgeneric`/`defmethod` を使います。

## クラス
```lisp
(defclass point ()
  ((x :initarg :x :accessor point-x)
   (y :initarg :y :accessor point-y)))
```

## インスタンス
```lisp
(defglobal p (make-instance 'point :x 3 :y 4))
(point-x p) ; => 3
```

## ジェネリック関数
```lisp
(defgeneric norm (obj))
(defmethod norm ((obj point))
  (+ (* (point-x obj) (point-x obj))
     (* (point-y obj) (point-y obj))))
```

## 注意
メソッド選択規則（多重ディスパッチ・曖昧性処理）は処理系実装に依存する詳細があります。
