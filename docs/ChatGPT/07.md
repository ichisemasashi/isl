# 7. オブジェクトシステム

## 対応節
- §10 Classes
- §15 Objects

## 7.1 クラス定義
```lisp
(defclass point ()
  ((x :initarg :x :accessor point-x)
   (y :initarg :y :accessor point-y)))
```

## 7.2 インスタンス化
```lisp
(defglobal p (make-instance 'point :x 3 :y 4))
(point-x p) ; => 3
```

## 7.3 ジェネリック関数とメソッド
```lisp
(defgeneric dist2 (obj))
(defmethod dist2 ((obj point))
  (+ (* (point-x obj) (point-x obj))
     (* (point-y obj) (point-y obj))))
```

## 7.4 継承と選択
```lisp
(defclass colored-point (point)
  ((color :initarg :color :accessor point-color)))
```
メソッド選択規則の詳細（曖昧性など）は、規格要件 + 処理系仕様で確認する。
